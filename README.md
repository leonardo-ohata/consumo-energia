# Calculadora de consumo de energia 
Este projeto foi criado para praticar o ciclo **Entrada** -> **Processamento** -> **Saída** do Python. 
O programa pede o **nome do aparelho**, sua **potência em watts** e seu **tempo de uso médio em horas**. 

Então calcula-se a **média de consumo** por meio da seguinte fórmula: **potencia** * **tempo** * 30 /1000.

Após isso é calculado o **custo mensal** por meio da fórmula: **média de consumo** * **preço do kWh** que, neste caso, o escolhido como base foi o da cidade de São Paulo (0.72). 

Para executá-lo corretamente são necessárias algumas dicas: 

Quando forem pedidas informações como: **potencia** e o **tempo**, é necessário que seja respondido somente com números sem o acompanhamento de suas unidades de medida. Caso o contrário, ele não reconhecerá  os valores e dará erro.
